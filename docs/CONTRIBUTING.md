# Contributing Guide

## How to Contribute

1. Fork the repository.
2. Create a new branch for your feature.
3. Commit your changes.
4. Push to the branch.
5. Open a Pull Request.

## Adding Content
- Add new jobs in `frontend/content/en/jobs/`.
- Add new skills in `frontend/content/en/skills/`.

## Developing Backend
- Add new endpoints in `backend/api`.
- Run tests in `backend/tests`.
